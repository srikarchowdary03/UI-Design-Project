Computer Prototyping

The Computer Prototyping page serves as an interactive representation of our design. This prototype includes the implementation of major functionalities, showcasing how the user can interact with the platform.

Link to Computer Prototype:

Open the Computer Prototype in Figma

Instructions to Run the Prototype:

  • Click the link above to open the Figma prototype.
  • Navigate the Prototype using the navigation bar on the top to switch between different pages (e.g., Home, Transportation, Services, Academics).

Features to Explore:

  • Home Page: Utilize the navigation bar to move between various sections.
  • Transportation: Click to access the Transportation page.
  • MBTA on the Transportation Page: Select this option to view the Subway Lines page.
  • Services: Click to navigate to the Services page.
  • Academics: Click to open the Academics page.
  • Tutorials Section on the Academics Page: Select this section to explore tutorials.
