Instructions to Explore the Website:

  • Click the link above to access the live version of B-town.
  • Use the navigation bar at the top of the website to explore various sections:
    • Home: The starting point of your journey.
    • Transport: Detailed information about Boston's transport system.
    • Events: Discover events happening in Boston.
    • Subway: Real-time schedules for different subway lines.
    • Services: Access city services like health and safety resources.
    • Academics: Explore libraries, bookstores, and tutorials.

Features Implemented:

  • Dynamic Navigation: Seamless navigation across multiple sections.
  • Search Bar: Use keywords to quickly access specific sections (e.g., "Red Line," "libraries").
  • Event Filters: Refine events by categories like "Music," "Sports," or "Festivals."
  • Responsive Design: Optimized for devices of all sizes.
  • Interactive Features: Dynamic pages for Subway schedules, Events, and more.

For Developers:

  • Clone the repository: git clone https://github.com/srikarchowdary03/UI_Design_B-Town.git
  • Navigate to the project directory: cd UI_Design_B-Town
  • Install dependencies: npm install
  • Run the development server: npm start
  • Access the application locally at http://localhost:3000.
